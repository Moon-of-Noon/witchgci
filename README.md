## 简介

夏织的生活轨迹被一场突如其来的大事件所扭转。

新的世界、新的朋友、新的能力……一切都是以往那副凡俗之身所无法触及的梦幻。

唯有夜明梦醒，意识到自己再也回不到从前，两种矛盾的身份互相撕扯，让夏织不由得询问自己：我是谁，我在哪儿？晚上吃什么？

光阴荏苒，新生的小魔女久违地回到自己被毁灭的故乡，拉了拉身旁的友人，做出一副饿狠狠的模样：“你知道吗，我想把你的脑袋当夜壶使。”

友人只是捋了捋夏织的长发，哂笑着：“那就角斗场上既分高下也分生死罢，但首先，你得能打赢我。”
