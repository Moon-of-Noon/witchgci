白百合晃晃悠悠地坐上了书桌前的椅子，以她的身高来说，这个椅子还是有些高的。

“前几天我的两位母亲带我去外面打猎了，就用的步枪，我只用了三颗子弹就制服了那只傻狍子。”谈到开心的事情，白百合身后的翅膀也跟着微微颤动起来，像是想要表达什么喜悦的情绪。

“我嘛……昨晚上做了个噩梦，然后醒来的时候发现房间里多了一只超大的蜘蛛标本，这也许算是某种猎物？”夏织指着依然摆在房间里的那只眠梦蛛遗体。

巨大的蜘蛛相当显眼，白百合刚进来就看见它了，现在听到是夏织的战利品更是冒出了星星眼。

“好厉害呢。不管是什么动物，只要到了一人大这个体格都会变得很难制服。但是为什么它甲壳的上半部分消失了呢？”

夏织当然不知道为什么这只蜘蛛的甲壳会莫名其妙消失一部分，于是选择了转移话题。

她从书架上抽出一本故事书，放到了白百合手上：“我们一起看书好吗？”

“好耶~”

与此同时，一楼客厅中，白雪辉正看着桌上摆着的那一盘克兰西亚瓜，那是西芙奈莲的回礼。

魔力雷达告诉她，面前这一小盘土里土气的东西是极其稀有的能够增长魔力的奇遇物品。哪怕只能增加几十点魔力上限，它的价值也远超自己带来的那些甜品本身。

魔女很少会不遵守社交礼仪，又或者是弄错已知物品的价值，赠礼与回礼之间巨大的价格差让白雪辉陷入了不安之中。

“西芙奈莲女士，你没弄错吧？”

“不必惊慌，这些就当作我对白百合的奖励吧，能让夏织在学校里交到第一个朋友的奖励。”

“可是，这太贵重了……”白雪辉自然是能分清楚同学之间送点小点心和送贵重物品的区别的。而且，那种事情似乎只是举手之劳吧。

在西芙奈莲的强硬要求下，白雪辉不情不愿地收下了这些礼物，准备去看看两只幼魔女们在干什么。

“咚咚咚。”

“请进。”

夏织回过头去，看见了站在前面的西芙奈莲与后面跟着的一行人。

原来是在一起看书啊，这关系确实是真的很紧密了。白雪辉搓了搓夏织的脑袋，惹得夏织畏畏缩缩往西芙奈莲那边靠了一点。

察觉到夏织不太愿意接受这种亲切交流，白雪辉的注意力转向了自家小魔女身上：“小百合，我们该回家了。”

“可是我还没看完这个故事呢……”

虽然有些遗憾，但白百合依然顺着自己母亲的意思站了起来，略有不舍地走了。

夏织本想颇为大度的把这本书借给白百合看，但想到西芙奈莲曾经说过的社交礼仪，自认为两个人的关系还没那么密切，只得作罢。

能把自己的书库给其他魔女看就已经是一种非常大度与关系亲密的表现了。更进一步，能做到把自己的书借阅给其他魔女的话……至少也得是同穿一条裤子的那种等级了。

希望我们九月份开学的时候还能再见，白百合同学。

或许是有点害怕离别时的场景，夏织一直未能把这句话真正说出口，哪怕它寻常的就跟一句“再见”一般。

明明已经不是陌生人了呢……想到这里，她的内心变得更加烦闷了。

夏雪注意到了夏织的情绪有些不太好。

她经常这样呢，和别人在一起玩的时候会变得很开心，要到回家时间了就变得特别烦闷。

那么就让我来陪着她出去玩闹一会吧，这也许可以改善一下她的心态。

夏雪用指背刮擦着夏织的翅膀，看到她有所反应后，停下了手，发出了邀请：“出去转一转吗？”

“呜……行吧，就当去吹吹风了。”

总感觉她很不乐意的样子呢。

但夏雪很清楚夏织这会的别扭心态得反着理解，她其实还是很想有人能陪着的，只是心境还沉浸于之前的悲伤之中没转变出来而已。

跟西芙奈莲说清楚要带着夏织出去玩之后，夏雪来到了停鸡坪旁，拉着夏织一起坐上了扫帚。

翱翔在高空之中，原本阴郁的心情也随着看见了地上的草木而变得轻松起来。

这里是云道，空中飞行的主干道，现在正是交通繁忙的时候，大量急着赶路的魔女正在快车道上一路狂奔，我们的夏雪小姐则是坐着自己的儿童款扫帚，以一个慢悠悠的二百千米每小时的速度在慢车道上晃荡着。

一旁也有不少一样不急着赶路的魔女，一边飞着还一边有说有笑地互相交流着，偶尔还能看见一辆巴士自公交车道上穿行而过。

“住独门小空岛就是这点不好，想出门坐巴士还得先走到附近的大型空岛群才能等到班次。要把扫帚寄存起来还得付一块钱停车费呢，不如我自己骑过去了，可惜我不认路啊……”夏雪有一遭没一遭地闲谈着。

她的生活也不止家与学校两点一线。偶尔有一些实战课得去别的空岛群上课，又或者是几位好友互相约个地方吃饭的时候，公共交通就成为了首选。家门口坐不到公交车成了夏雪最大的怨念——拜路痴体质所赐，她自己骑扫帚出去九成九找不到路，快两个月了还是只记得家到学校怎么走。

但对夏织来说，平常出门都有西芙奈莲开车带着，自然是没有这种烦恼的。

“说回来，你应该还没试过吧，要不今天我就带你坐一次巴士，体会体会这个世界的狂野玩意。”

面对夏雪的提议，夏织有些心动了。

“得嘞！”

夏雪在前方一个路口调转了方向，朝着不远处的大型空岛奔去。
