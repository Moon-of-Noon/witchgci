魔女世界的公共交通，常见的大概可以分为短途的巴士与长途的飞艇，除此之外还有在繁华地段开行的幽灵列车又或者是别的什么东西。有些地区甚至会根据当地特色开发出新式交通工具，譬如说在曾经的戊易太上黄庭的领地中便流行着仙人腾云驾雾的传说（一辆小型家用筋斗云售价仅需3999元！），一度成为知名旅游景点。

“滴！目标站点：酒仙岭酒湖区；乘车人数：二人。请支付叁点零零元！”

自动售票机中传出了不知道哪位魔女亲自配音的阴阳怪气的声音，欠揍的想狠狠地殴打一顿这个机器。

这是南城公共交通公司最近才换上的音频。据说在第一天使用时就传出了不下两位数的售票机损坏的事件，以至于不得不临时增派人手进行人工售票服务以及保护珍贵的售票机不被损坏。

夏雪向售票机中投入了三枚硬币，取出了刚刚印刷好、还散发着油墨香气的车票，把其中一张递给了夏织。

“拿好你的车票，上车的时候在闸机上打个孔，下车的时候也要在闸机上打个孔，知道了吗？”

这车票听着就跟曾经的纸质火车票一样，甚至票根还能自己收藏起来，夏织突然间联想到了自己在克兰西亚的那个家之中留着的一些东西。

那些都是曾经的我留下的宝贵记忆呢，她摇了摇头。

现在只需要去记忆殿堂里翻阅对应的资料就好了，能做到完全潜入式虚拟实景体验，就连当时的神态与心情都记忆得一清二楚，甚至还能利用一下梦境的规则，直接复刻一个“真的”仿制品。

那么，承载了那些记忆的道具又有什么用呢？

机械性地走到站台前，夏织瞥见了地上的黄线，小腿自己停了下来，但思维依然停留在对那些遗物价值的思考中。

它们证明了我去过什么地方？见证了我的旅途与悲喜？又或者是曾经的那个我与现在的我的分割线？

〔有趣的想法，我整理了这么久的记忆，一开始也只是为了整理清楚，就像是强迫症一样。至于它有什么意义，值得我也思考一下。〕

张可颂的思绪从深处传来，很快又陷入沉寂。

抱歉撒，现在不能去找你一起探讨这个问题，张可颂同学。

〔其实你的所思所想我都知道，所以即使是“醒着”的时候，我们也是可以正常交流的。〕

悠长的汽笛声响起，夏雪拽了一下夏织的胳膊，将她从思索中拉出来：“车来了，我们该上车了。”

欸？这么快吗？总感觉刚才是不是错过了什么东西？

按照指示把票据插入闸机之中，清脆的剪纸声响起，原本关着的闸机打开了，夏织先行进入到了飞艇的内部。

夏雪又拉着夏织坐到了一组特别宽大的位置上。

“亲子座位……我们真的要坐在这里吗？”

尽管很早就认命自己的年龄不幸归零这件事，夏织依然有些讨厌这种暗示她身份的地方。

“至少在这里你可以放心地把翅膀展开而不担心打到什么东西就是了。每次看着你把翅膀窝成一圈我都感觉……幻肢痛。”夏雪又抓了一把夏织翅膀上的绒毛。

夏织倒是觉得把翅膀折成披肩的样子会更舒服一些，毕竟完全展开开来的话真的太沉了，很容易就腰酸背痛什么的。

但借着这个机会偷偷薅一把毛可就说不过去了，夏织展开了羽翼，装模作样地给夏雪扇了一下，同时逗乐了两个人。

“呜————”

一阵长鸣笛后，巨大的空艇合上了门，从地面上漂浮起来，以不合体态的灵敏度原地自转九十度，回到了正常的航线上。

这个过程也很温和，夏织没有感觉到加速度的存在，也没感觉到要被甩出去的那种感觉。

从观察窗向外看去，身后的空岛正迅速变小、消失成一个点。

好快呢，比骑扫帚快多了呢。

在中途停靠了两个站点以后，飞艇停靠在了酒湖空岛上。周围的云雾浓郁地仿佛要凝聚成实体，散发着浓浓的酒香。

夏织从站点中走出来，闻到了极度浓郁的酒气，辛辣的感觉刺挠着她的鼻子，咳嗽得差点没呕出来。

一位在这里执勤的保安看见了初来乍到又显得极不舒适的夏织，抬起手上的法杖，释放了两个小法术。

无形的气泡升起，包裹住了夏织的头部，隔断了内外空气的循环，又在配套的制氧法术的配合下生成了清新而柔和的空气，缓解了“用鼻子喝酒”的不适感。

终于舒服一点了……

夏织抬起头，看向此处熙熙攘攘的行人，像是自己一样的小魔女是几乎见不到的。

为什么呢？

总不可能是未成年魔女不得饮酒吧。

不过这里确实有其他的小魔女存在，看来应该不是这些酒气云的问题。

“喔，你没事儿吧。这里的特色就是蕴藏于云雾中的酒精，我的学校也在这里哦。”

夏雪后知后觉地把夏织抱了起来，轻轻拍着她的后背。

夏织张开了左侧的翅膀，围住了夏雪的脖子。

“好了，我们现在去坐巴士，带你去看大熊猫。”夏雪抱着夏织往巴士站走去。

这里也有大熊猫吗？

大熊猫喝醉了会不会打醉拳？或者是表演一个《功夫熊猫》出来？

像是要解答夏织的疑问一样，夏雪又补充了一句：“它们的拿手好戏可是喝下一大口酒然后喷火！”

在路上，夏雪自顾自的又说了许多她上学时听到的酒仙岭见闻，但夏织只是兴致缺缺地左耳朵进右耳朵出而已，心思全被会喷火的熊猫吸引走了。
