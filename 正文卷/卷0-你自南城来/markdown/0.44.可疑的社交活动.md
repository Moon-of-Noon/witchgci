今天的天影学园格外的拥挤，许多魔女家长亲自将孩子送入了教室之后还要站在窗外观察。即使走廊足够宽阔，一堵人墙杵在那里也是极其壮观的。

有些奇怪呢，以往家长们不都是很放心地让孩子自己走入学校里吗？

西芙奈莲牵着夏织的手，沿着走廊走到了教师办公室的另一个入口，找到了之前的李蓉老师。

“西芙奈莲，今天是个什么日子啊？”对此疑惑不解的夏织选择了直接提问。

“是你体检的日子，每个月的第一个周一。你的同学们或许正在高兴于今天能少上半天课吧。”

能少上半天课的话，那确实很开心了，或者说只要是该上课的时候能够不上课就会很兴奋。

对夏织来说，上学这件事尚且还处于一种新鲜的感觉之中，还没有被学业摧残的她自然是感觉不到那种兴奋的感觉的。

李蓉将一份表格递给了西芙奈莲，上面还有一些内容需要让家长确认与签字。

上面所写的体检相关项目和上个月所做过的相差不大。西芙奈莲仅仅是扫视了一眼，左手上的一根法杖轻轻一点，一个有着法律效应的魔法印记便拓印到了纸上。

李蓉收回了表格，示意二人可以离开这里了。

教室里一如既往地跟一万只鸭子在嘎嘎叫一样。以往都是上课铃敲响了以后才会变得安静，而今天的下午时刻没有课程，自然不会敲上课铃。

夏织走进了教室，从书架上取下一本故事书，再从一旁的板凳堆中搬出一个凳子，摆到墙边，品味着魔女版本的童话故事，好似另一边那十几位小魔女不存在一样。

〔为什么不去试试结交几个朋友呢？等到了九月份正式开学了之后，她们都还会是你的同学的，明明你说了你想去改变什么的。〕

张可颂的声音又响了起来，让夏织好一阵尴尬。

是啊，明明我出门前还在想着能不能改变一下自己的。

看见了这个班级中唯一一个身后长着翅膀的同学来了之后，雷杰从她交流得正欢的小群体中脱离了出来，小跑到了夏织的身边。

再一再二不再三，咱都这么坚持了，应该可以打动她了吧。如果要是还是失败了的话，我就明天再试一次，总会让她愿意把翅膀给我摸摸的！实在不行……那就忍痛割爱一次，用家里秘制的甜点去打动她！

定下了新三步走计划的雷杰用着前两日刚才家里人处学来的“为人处世的小技巧”向夏织发起了交流。

“这位姐妹，我是雷杰，请问你的名字是什么？”

夏织的注意力从书本上离开，看向了面前这位略带婴儿肥的同学。

尽管从记忆中翻出来了前几天并不太友好的相处部分，但我如果要继续甩着一副臭脸的话也的确对不起她了，至少需要礼节性的回应一下。

那么，社交礼仪的第一步是……直视对方的眼睛，然后说出自己的想法。

夏织的瞳孔有些躲闪，她并不是很适应这种目光对视的感觉，却坚持着看着雷杰眼瞳中自己的倒影。

“我，我叫夏织哦…”

声音有些小了，但我做到了呢，虽然有些不太完美。这在以前只有西芙奈莲才能让我如此放心的这么做。

很好，对方回应了我的请求，这说明她至少有着与我交流的想法。那么现在，我该提出另一个请求了。

雷杰顺着计划的第二步提出了另一个问题。

“你的翅膀很好看呢，漆黑的闪烁着流光的色泽，想必平时肯定保养得很好吧。那么，能让我看一下全貌吗？”

先通过夸赞让对方心情变得愉快一些，这样再提出小要求的时候就更容易被同意。而且，我只要求比较简单的事情，把一件大事儿像是切香肠一样拆分开来，慢慢削弱对方的心理防线……虽然我的父族母亲说这样做小心被发现了之后被人套麻袋打一顿就是了。

夏织在听见对方提出来的要求的那一刻直接进入到了恐慌之中，就连原本服服帖帖扮作披肩下摆的羽毛也跟着炸了起来。

她想看我的翅膀是想做什么？想像是上次那样直接强行拽过去薅吗？

在紧张之间，夏织的魔力感知网不自觉地向外蔓延，触碰到了窗台边的西芙奈莲。一股安心的感觉又一次涌上了心头。

对，就是这样，至少情况不会变得太糟糕了，我还是有后路的！

张可颂的声音也在一两秒的延迟之后传进了夏织的思维之中。

〔有点熟悉的做法，注意一下对方是否存在切香肠战术的可能，你可以试着配合一下。如果还有下一个要求的话，直接拒绝她就好了，态度坚决一点。〕

张可颂也这么说了，那我可就要试一试了。

身前交叉着的羽翼慢慢展开，停留在了空中。即使有些刚刚因为恐慌而炸出来的羽毛还没有被抚平，依然让雷杰流露出了渴求的面容，好似下一秒就要直接扑上去一样。

果然呢，她的想法不只局限于“看一眼”。

但雷杰也注意到了那些根根竖起的羽毛，结合上夏织明显没掩盖住的容貌，只怕是做出这个选项就已经让她陷入到接近暴躁的程度了。

那么，计划暂停进行，剩下的工作留到明天，又或者是后天？总之需要给她留一个缓冲时间，让她慢慢放下那些防备。

尽管很想趁着这个机会赶紧抓一把就跑，但雷杰忠实地遵守了自己父族母亲所制定的计划，避免真的让夏织再次一翅膀给自己拍飞两米远。

看着走远了的雷杰，张可颂满是不在乎地评价着：“看起来她应该是从哪儿学到了这么一招，只是没算到你其实还有一个外置大脑的存在，而且手法其实相当的粗糙。之后的话，她还是会这样跟你不断地套近乎，想办法让你认为她会是个好朋友，然后干出……这样或者那样的事情，小心了。”

“就没有什么一劳永逸解决这种人的办法了吗？”

“很简单，一翅膀拍上去就行了。她不会自讨没趣的。”

“我感觉那可能是在奖励她。”

张可颂撇了撇嘴，又窝进了意识空间的床铺中，代表着自己暂时不想被打扰。

雷杰也并不知道，如果自己一上来就献出甜品的话，这会的夏织应该早就被撸得舒服的呼噜叫了。

小魔女的本性总归是贪吃的，但也正是因为这一点导致她错失了良机。
